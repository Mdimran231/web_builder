# 🚀 WebBuilder - AI-Powered Website Generator

**WebBuilder** is a high-performance web application designed to automate the website creation process. By leveraging the **Google Gemini 1.5 Flash SDK**, it transforms natural language prompts into clean, production-ready HTML and Tailwind CSS code in seconds.

## 🌐 Live Demo
Check out the live application here: [https://imran-builder.netlify.app](https://imran-builder.netlify.app)

## ✨ Key Features
* **Real-time AI Generation:** Integrated with Google Generative AI to generate structured, semantic code.
* **Live Preview Window:** Features a dedicated iframe for instant rendering of the generated website.
* **Modern UI/UX:** Built with a sleek **Glassmorphism** interface and a fully responsive layout.
* **Tailwind CSS Integration:** Automatically applies utility classes, ensuring modern and scalable styling.
* **Developer Friendly:** Generates portable code that can be easily copied and used in any project.

## 🛠️ Tech Stack
* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS
* **AI Engine:** Google Gemini 1.5 Flash (Generative AI SDK)
* **Deployment:** Netlify (Continuous Deployment)
* **Version Control:** Git & GitHub

## 🚀 Installation & Local Setup

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Mdimran231/web_builder.git](https://github.com/Mdimran231/web_builder.git)
   cd web_builder
Install Dependencies:

Bash
npm install
Environment Configuration:
Create a .env file in the root directory and add your API key:

Code snippet
VITE_GEMINI_API_KEY=your_actual_gemini_api_key
Run the Development Server:

Bash
npm run dev
📝 Configuration Note
For production deployment on Netlify, ensure you add the VITE_GEMINI_API_KEY under Site Settings > Environment Variables so the AI engine can authenticate requests.
